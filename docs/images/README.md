# Інструкція для додавання скриншота

1. Зробіть скриншот web інтерфейсу FiatLux (http://localhost:3000)
2. Збережіть файл як `screenshot.png` в цю папку
3. Рекомендований розмір: ширина 1200-1600px для якісного відображення
4. Формат: PNG або JPG

## Що має бути на скриншоті:

- Головна сторінка з таблицею графіків
- Видно годинник (0-23) по горизонталі
- Видно черги (1.1, 1.2, 2.1, 2.2...) по вертикалі
- Кольорове кодування (червоний/зелений/жовтий)
- Випадаюче меню з вибором черги (якщо є)
- Обидва графіки: поточний та майбутній

## Після додавання скриншота:

```bash
git add docs/images/screenshot.png
git commit -m "Add web interface screenshot"
git push
```
